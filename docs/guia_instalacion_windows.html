Guía de Instalación
Hermes Word Add-in en Windows

Paso a paso — panel de chat AI dentro de Microsoft Word

¿Necesito instalar Hermes Agent?

NO. El backend (backend_server.py) es un servidor Python autónomo que llama directamente a la API del LLM que elijas (DeepSeek, OpenAI, Anthropic, Ollama local, etc.). No requiere Hermes Agent instalado.

Pasos de Instalación

  1. Instalar Python en Windows

    Descargar de python.org, ejecutar el instalador, y marcar "Add Python to PATH". Verificar:

    python --version

  2. Clonar el repositorio

    cd C:\Users\TU_USUARIO\
git clone https://github.com/mmansillaf/hermes-word-addin.git
cd hermes-word-addin

    Si no tenés Git: git-scm.com/download/win

  3. Instalar dependencias Python

    pip install fastapi uvicorn websockets

  4. Configurar proveedor LLM y API Key

    Elegí tu proveedor:

    DeepSeek (recomendado, mejor relación costo/calidad):

    $env:LLM_PROVIDER="deepseek"
$env:DEEPSEEK_API_KEY="sk-tu-key"

    OpenAI:

    $env:LLM_PROVIDER="openai"
$env:OPENAI_API_KEY="sk-tu-key"
$env:LLM_MODEL="gpt-4o"    # opcional, default: gpt-4o-mini

    Anthropic Claude:

    $env:LLM_PROVIDER="anthropic"
$env:ANTHROPIC_API_KEY="sk-ant-tu-key"

    Ollama local (gratis, sin internet):

    $env:LLM_PROVIDER="openai-compatible"
$env:LLM_BASE_URL="http://localhost:11434/v1/chat/completions"
$env:LLM_MODEL="llama3.1:8b"
$env:LLM_API_KEY="ollama"

    Groq / Together / vLLM (cualquier endpoint OpenAI-compatible):

    $env:LLM_PROVIDER="openai-compatible"
$env:LLM_BASE_URL="https://api.groq.com/openai/v1/chat/completions"
$env:LLM_API_KEY="gsk-tu-key"
$env:LLM_MODEL="llama-3.1-8b-instant"

    Para que persista entre reinicios:

    Win+R → sysdm.cpl → Avanzadas → Variables de entorno → Nueva variable de sistema

  5. Lanzar el backend

    cd src
python backend_server.py --port 8765

    Debe mostrar: Listening on http://localhost:8765

    ⚠️ No cierres esta ventana. El servidor debe correr mientras usás Word.

    Probar sin Word: abrí http://localhost:8765 en tu navegador.

  6. Instalar el Add-in en Word

    6.1 Habilitar Programador: Archivo > Opciones > Personalizar cinta > marcar Programador

    6.2 Cargar manifest: Pestaña Programador > Complementos > MIS COMPLEMENTOS > Cargar mi complemento > Seleccionar manifest.xml de la carpeta src/

    6.3 Abrir panel: Click en botón "Hermes AI" en pestaña Inicio

  7. Usar el Add-in

    BotónQué hace
    Leer docCarga el texto del documento activo
    AnalizarHermes analiza estructura, métricas, sugiere mejoras
    ResumirGenera 3-5 bullet points del documento
    ReescribirMejora claridad y profesionalismo
    Insertar en docInserta la respuesta al final del documento

    También podés escribir consultas libres y presionar Enter.

Solución de Problemas

"No se pudo cargar el complemento"

Verificá que el backend está corriendo (Paso 5). Abrí http://localhost:8765 en el navegador. Cerrá y reabrí Word.

"Failed to fetch" en el panel

El backend no responde. La ventana de PowerShell debe seguir abierta con el servidor. Verificá el puerto con netstat -an | findstr 8765.

El texto no se inserta en el documento

El documento debe estar en modo edición. Verificá que no esté protegido o en solo lectura.

Error de API key

La variable de entorno de tu proveedor no esta configurada. Revisa el Paso 4 segun el proveedor elegido.
